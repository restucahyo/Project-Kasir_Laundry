🧺 Laundry Cashier System (Python + MySQL)

Aplikasi kasir sederhana menggunakan Python dan database MySQL. 📌 Fitur

* Input data pelanggan
* Hitung harga laundry
* Simpan transaksi ke database
* Menampilkan data transaksi

 ⚙️ Teknologi

* Python
* MySQL
* mysql-connector-python
 🛠️ Cara Menjalankan

 1. Install library

pip install mysql-connector-python

 2. Setup database

* Gunakan MySQL (Laragon / XAMPP)
* Buat database baru
* Import file `database.sql`


3. Jalankan program

python laundry.py

📌 Catatan
Project ini masih berbasis terminal (CLI) dan bisa dikembangkan menjadi aplikasi GUI atau web.
